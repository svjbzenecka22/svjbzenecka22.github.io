# SVJ Bzenecká 22 Web

Pracovní projekt pro sjednocený web SVJ Bzenecká 22, Brno.

## Aktuální cíl

Vytvořit jeden hlavní web, pod který budou postupně zařazeny tematické sekce a projekty SVJ.

Aktuálně jsou převzaty tyto existující části:

- `shromazdeni_260427/`
- `balkony/`

## Struktura projektu

- `index.html` - hlavní rozcestník webu
- `shromazdeni_260427/` - existující sekce ke shromáždění vlastníků
- `balkony/` - existující sekce k projektu balkónů
- `docs/working-log.md` - průběžný záznam rozhodnutí, otevřených otázek a navazujících kroků

## Poznámka k pokračování

Další práce má nejprve proběhnout formou analýzy budoucích potřeb webu. Teprve podle ní se rozhodne,
zda celý projekt zůstane statický na GitHub Pages, nebo zda některé neveřejné části budou řešeny
oddělenou aplikací.
